## 更改日期型別
```SQL
ALTER TABLE world
ALTER COLUMN 日期 TYPE DATE
USING 日期::DATE;
```

```SQL
SELECT 洲名,國家,日期,總確診數,新增死亡數,總人口數
FROM world;
```
